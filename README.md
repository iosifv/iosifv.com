- [Upstash](https://upstash.com)
- [Contentlayer](https://www.contentlayer.dev/)
- [Vercel](https://vercel.com/)

<br/>

## Running Locally

Create a `.env` file similar to `.env.example`.

Then install dependencies and run the development server:

```sh-session
git clone
pnpm install
pnpm dev
```
